EgretGameEngine

EgretGameEngine 是一套基于Egret引擎封装的业务库，提供了业内丰富的封装业务能力，大大降低了一个初创团队前期对业务积累需求。

快速使用

进入github的仓库地址，下载引擎代码，或者使用git clone 对项目进行克隆到本地

git clone https://github.com/yicaoyimuys/EgretGameEngine.git

下载后解压安装包会看到Egret的基本项目目录。

使用Egret Wing 对代码进行编译，运行后能够看到以下的画面恭喜你已经成功运行了本引擎的代码。
